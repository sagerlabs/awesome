## Cases

### 2026-05-09 · Seeded Rejected Samples

这些样本来自本地试玩与人工复盘，用于把 ADR-010 的 feedback loop（反馈闭环）从“只有代码”推进到“有可回放案例”。后续可以继续追加真实用户样本。

1. 日期：2026-05-09
用户原问题：开局女神之泪，我能玩什么？
Agent 原回答摘要：推荐裁决使铁男/滑板鞋、法师球女/维鲁斯、夜幽锐雯。
用户反馈：不是当前版本信息。
判定类型：advice_rejected
可能原因：Data / Prompt
后续修复方向：回答必须引用当前 knowledge 命中的版本和阵容，禁止用模型记忆补旧赛季。

2. 日期：2026-05-09
用户原问题：当前版本最强的三套阵容是什么？
Agent 原回答摘要：输出迅箭永恩、护卫神谕卡莎、夜幽锐雯及疑似旧版本数据。
用户反馈：第一条就败了。
判定类型：advice_rejected
可能原因：Data / Query
后续修复方向：Top comp 查询只允许从 team_comps 命中结果排序，不允许编榜单。

3. 日期：2026-05-09
用户原问题：当前版本最强的三套阵容是什么？给我平均名次、前四率、吃鸡率和核心装备。
Agent 原回答摘要：输出 `TFT16_Augment_RumbleCarry` 等技术 ID。
用户反馈：回复不地道、不优雅。
判定类型：advice_rejected
可能原因：Prompt / Data
后续修复方向：对外回答必须展示中文阵容名，技术 ID 只能作为调试字段。

4. 日期：2026-05-09
用户原问题：我有珠光护手，可以玩什么阵容？

用户反馈：回答太勉强。
判定类型：advice_rejected
可能原因：Query / Prompt
后续修复方向：装备匹配要区分“核心装备”和“可消化装备”，弱关联不能强推。

5. 日期：2026-05-09
用户原问题：我有兰博、凯南、璐璐，还有羊刀，能冲吗？
Agent 原回答摘要：先声明数据来源，再给约德尔人兰博。
用户反馈：应该先肯定结论，数据来源放最后。
判定类型：advice_rejected
可能原因：Prompt / UX
后续修复方向：回答结构改成“结论先行 -> 理由 -> 操作 -> 数据来源”。

7. 日期：2026-05-09
