# TFT阵容推荐优先级优化

## 任务概述
根据用户输入的英雄和装备，实现新的阵容推荐优先级规则

## 需求拆解

### 1. 英雄优先级 > 装备优先级

### 2. 通过英雄选择阵容的优先级
- 筛选强度最高的三套阵容
- 根据英雄在阵容中的优先级再排序（是否为主C）
- S16赛季解锁机制：英雄能解锁高费卡 → 优先选择解锁后的S梯队阵容
- 解锁后无S梯队 → 只根据当前英雄筛选三套

### 3. 通过装备选择阵容的优先级
- 无英雄输入 → 主C刚需该装备的最强三套
- 有英雄但无S梯队 → 主C刚需该装备的最强三套
- 有英雄且有S梯队 → S梯队刚需该装备的最强三套，与英雄筛选结果合并排序（英雄 > 装备S梯队）

### 4. 羁绊梯队索引
- 按阵容等级排序 → win_rate → top4_rate
- 按羁绊在S梯队中出现的质量排序

## 实现计划

1. 在 `tft/tool/` 下创建新文件 `priority_recommend.go`

2. 实现函数清单：
   - `SelectCompsByHeroWithPriority()` - 按英雄优先级选阵容
   - `SelectCompsByItemWithPriority()` - 按装备优先级选阵容
   - `BuildTraitTierIndex()` - 构建羁绊梯队索引
   - `MergeAndSortRecommendations()` - 合并和排序推荐结果

3. 修改或新增类型定义（如需要）

4. 更新相关调用链
